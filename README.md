# 電影時刻表
### 簡介
查看3家電影院的即時電影時刻表，以及電影介紹
### 使用技術
- c# 撰寫爬蟲程式，將時刻表資訊存入MS SQL
- ASP.NET MVC 
- 套用Bootstrap TheEvent版型
- ajax 更新查詢畫面
- 套用 jQuery UI 日期選擇器（Datepicker）來選擇日期
